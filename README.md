## 選択モード変更
### 運ゲーモード
**最後の試合が観戦者ではなかった人**から観戦者を選択します。
連続した観戦の可能性は無いですが、__**参加数に開きが出てしまう可能性があります。**__

### 平等にモード
**試合参加数が多い順**に、観戦者を選択します。
参加数を平等にしやすいモードですが、新規が入ってきた場合、
__**連続した観戦の可能性が出てきます。**__
